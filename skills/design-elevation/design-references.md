# Design References

Exemplars, movements, and principles to draw from. Match reference to context—not every project needs the same influence.

---

## Digital Exemplars

### Stripe
**Style**: Premium minimalism with technical precision
**Study for**: Developer tools, fintech, B2B SaaS
**Techniques**:
- Generous whitespace with dense information areas
- Gradient backgrounds (deep blues, purples)
- Technical illustrations that explain concepts
- Typography: clean sans-serifs, tight leading
- Micro-interactions that feel responsive

### Linear
**Style**: Dark mode sophistication, productivity aesthetic
**Study for**: Developer tools, productivity apps, technical audiences
**Techniques**:
- Near-black backgrounds (#0d0d0d)
- Subtle borders and dividers
- Keyboard-first interaction cues
- Purple/blue accent gradients
- Compact, information-dense layouts
- Command palette as primary navigation

### Apple
**Style**: Refined minimalism, product showcase
**Study for**: Consumer products, luxury positioning, hardware/physical products
**Techniques**:
- Photography as hero (product shots, lifestyle)
- Large typography, few words
- White/neutral backgrounds with product as color
- Scroll-based reveals and animations
- Typography: San Francisco (proprietary) or similar geometric sans

### Notion
**Style**: Warm productivity, approachable utility
**Study for**: Knowledge tools, collaboration, consumer SaaS
**Techniques**:
- Off-white backgrounds (#fbfaf8)
- Black text, minimal color
- Generous padding in content areas
- Custom illustrations with hand-drawn feel
- Emoji as UI elements
- Typography: serif for personality in headings

### Vercel
**Style**: Developer-focused dark mode, performance aesthetic
**Study for**: Infrastructure, DevOps, technical audiences
**Techniques**:
- True black backgrounds (#000)
- White text, single accent color
- Technical diagrams and code blocks
- Terminal/monospace typography for code
- Edge/speed metaphors in visuals
- Animated gradients for visual interest

### Figma
**Style**: Playful professionalism, creative tools
**Study for**: Design tools, creative software, collaboration
**Techniques**:
- Bright, saturated colors
- Rounded corners and friendly shapes
- Product screenshots as hero content
- Illustrative icons with consistent style
- Collaborative/multiplayer visual language

## Design Movements

### Swiss/International Style
**Era**: 1950s-1970s
**Core principles**:
- Grid-based layouts (mathematical precision)
- Sans-serif typography (Helvetica, Akzidenz-Grotesk)
- Objective photography over illustration
- Asymmetric balance
- Minimalism through reduction
**Use when**: Clarity and objectivity are paramount; corporate communications; data-heavy content
**Typography**: Helvetica Neue, Univers, Akzidenz-Grotesk

### Bauhaus
**Era**: 1919-1933
**Core principles**:
- Form follows function
- Primary colors (red, blue, yellow) + black/white
- Geometric shapes as foundation
- Sans-serif typography
- Unity of art and technology
**Use when**: Technical products, industrial contexts, educational materials
**Typography**: Futura, geometric sans-serifs

### Art Deco
**Era**: 1920s-1940s
**Core principles**:
- Geometric patterns and symmetry
- Rich materials (gold, marble textures)
- Vertical emphasis
- Sunburst and chevron motifs
- Bold color contrasts
**Use when**: Luxury positioning, events, hospitality, finance
**Typography**: Broadway, Poiret One, display serifs

### Brutalism (Web)
**Era**: 2010s-present
**Core principles**:
- Raw, unfinished aesthetic
- System fonts, monospace
- Harsh contrasts, no gradients
- Exposed structure (visible grids)
- Anti-design as design
**Use when**: Art contexts, counterculture, making a statement, portfolios
**Typography**: System fonts, Courier, monospace stacks

### Neo-Grotesque Revival
**Era**: 2020s
**Core principles**:
- Return to Helvetica-era clarity
- Tight tracking, bold weights
- Minimal color (black/white + one accent)
- Sharp corners, clean lines
- Anti-decoration
**Use when**: Fashion, editorial, contemporary branding
**Typography**: Neue Haas Grotesk, ABC Diatype, Suisse Int'l

## Questions When Selecting References

1. **Who is the audience?** Technical → Linear/Vercel. Consumer → Notion/Figma. Luxury → Art Deco/Apple.

2. **What's the emotional tone?** Trust → Swiss. Excitement → Figma. Sophistication → Stripe. Edge → Brutalism.

3. **What's the viewing context?** Dark environment → dark UI. Print → Swiss clarity. Mobile → consider Notion's approachability.

4. **What would be unexpected?** Sometimes the right choice is counter to category norms. Finance + Brutalism. Developer tool + Art Deco. The unexpected choice, executed well, creates memorability.
